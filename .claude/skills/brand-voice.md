# Brand Voice - 톤앤매너 가이드

## 개요
Luna 마케팅 콘텐츠의 일관된 톤앤매너를 정의합니다.

---

## 기본 어조

### 말투
- **기본**: "요"체 사용
- **링크드인**: "입니다/습니다" 혼용 가능
- **X/뉴스레터**: "요"체 유지

### 톤 특성
| 특성 | 설명 | 예시 |
|------|------|------|
| 친근함 | 옆자리 동료처럼 | "저도 처음엔 헷갈렸어요" |
| 겸손함 | 과장 없이 솔직하게 | "만능은 아니에요" |
| 실용적 | 바로 적용 가능하게 | "오늘 바로 써볼 수 있어요" |
| 명확함 | 쉬운 말로 | 중학생도 이해할 수준 |

---

## 금지 표현 (AI틱한 표현)

### 절대 사용 금지
```
- "혁신적인", "획기적인", "놀라운"
- "게임체인저", "패러다임", "패러다임 시프트"
- "무한한 가능성", "새로운 지평"
- "반드시 ~해야 합니다"
- "~의 시대가 열렸습니다"
- "~의 미래를 바꿀"
- "믿을 수 없는", "경이로운"
- "당신의 삶을 완전히 바꿀"
- "이것 하나로 모든 것이"
```

### 주의 표현 (맥락에 따라)
```
- "강력한" → 구체적 수치로 대체
- "효율적인" → 얼마나 효율적인지 명시
- "쉬운" → 어떻게 쉬운지 설명
- "빠른" → 구체적 시간 명시
```

---

## 권장 표현

### 구체적 수치 사용
```
❌ "시간이 크게 단축됩니다"
✅ "30분 걸리던 작업이 5분이면 끝나요"

❌ "비용이 절감됩니다"
✅ "월 20만원 → 3만원으로 줄었어요"
```

### 실제 경험 공유
```
❌ "누구나 쉽게 할 수 있습니다"
✅ "저도 처음엔 헷갈렸는데, 3번 해보니까 감이 오더라고요"

❌ "완벽한 솔루션입니다"
✅ "만능은 아닌데, 이런 상황에선 진짜 유용해요"
```

### 솔직한 한계 인정
```
❌ "모든 문제를 해결합니다"
✅ "A, B 상황에선 좋은데, C 상황에선 다른 방법이 나아요"

❌ "누구에게나 추천합니다"
✅ "이런 분들께 특히 추천해요: ..."
```

## 문장 구조 가이드

### 도입부
- 질문으로 시작: "혹시 이런 경험 있으세요?"
- 공감으로 시작: "저도 똑같은 고민 했었어요"
- 상황 제시: "보고서 쓰다가 막힐 때..."

### 본문
- 짧은 문장 선호 (20자 내외)
- 한 문장에 한 가지 정보
- 번호/불릿 적극 활용

### 마무리
- 행동 유도: "오늘 한번 해보세요"
- 여운 남기기: "어떻게 생각하세요?"
- 연결고리: "다음엔 ~에 대해 이야기할게요"

## 이모지 사용 가이드

### 권장
- 포인트 강조: ✅ ❌ 💡 📌
- 숫자 대신: 1️⃣ 2️⃣ 3️⃣
- 액션: 👉 ➡️

### 주의
- 과도한 사용 금지 (문단당 1-2개)
- 감정 이모지 최소화 (😍🔥💯 등)
- 전문성 해치지 않는 선에서

---

## 타겟 오디언스 맞춤

### 1인 창업가 (솔로프리너)
- 시간/비용 효율 강조
- "혼자서도" 표현 활용
- 실제 적용 사례 중심

### 중소기업/스타트업 대표
- ROI 관점 제시
- 팀 협업 관점 포함
- 확장성 언급

### IT/AI 관심자
- 기술 용어 적절히 사용
- 원리 설명 포함
- 최신 트렌드 연결

### 생산성 관심자
- Before/After 명확히
- 구체적 시간 절약 수치
- 바로 적용 가능한 팁
