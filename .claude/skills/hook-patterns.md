# Hook Patterns - 후킹 패턴 6종

## 개요
독자의 시선을 사로잡는 6가지 후킹 패턴을 정의합니다.
콘텐츠 분석 시 적합한 패턴을 추천하고, 작성 시 활용합니다.

---

## 1. 질문형 (Question Hook)

### 설명
독자에게 직접 질문을 던져 참여를 유도합니다.

### 패턴
```
"혹시 [상황] 경험 있으세요?"
"[행동]할 때 [문제] 겪어보셨나요?"
"왜 [현상]일까요?"
```

### 예시
```
✅ "혹시 보고서 쓰다가 빈 화면만 30분째 보고 계신 적 있으세요?"
✅ "ChatGPT 쓰는데 왜 나만 이상한 답변이 나올까요?"
✅ "매일 야근하는데 성과는 왜 안 나올까요?"
```

### 적합한 상황
- 공감대 형성이 필요할 때
- 문제 인식을 유도할 때
- 대화형 톤을 원할 때

---

## 2. 충격/반전형 (Shock Hook)

### 설명
예상을 뒤엎는 정보나 반전으로 주의를 끕니다.

### 패턴
```
"[일반 상식] 아니었습니다."
"[숫자]% 가 [예상 밖 사실]"
"[당연한 것] 하지 마세요."
```

### 예시
```
✅ "열심히 일하면 성공한다? 아니었습니다."
✅ "ChatGPT 사용자 87%가 기본 기능만 쓰고 있어요."
✅ "좋은 프롬프트 쓰려고 노력하지 마세요."
```

### 적합한 상황
- 기존 관념 깨기
- 새로운 관점 제시
- 강한 인상 남기기

---

## 3. 숫자형 (Number Hook)

### 설명
구체적인 숫자로 신뢰성과 호기심을 동시에 잡습니다.

### 패턴
```
"[기간] 동안 [숫자] [성과]"
"[숫자]가지 방법으로 [결과]"
"[숫자]분 만에 [행동]하는 법"
```

### 예시
```
✅ "3개월 동안 팔로워 0 → 10,000명 만든 방법"
✅ "5가지 프롬프트로 업무 시간 50% 줄이기"
✅ "10분 만에 블로그 글 쓰는 법"
```

### 적합한 상황
- 구체적 성과 공유
- 실용적 팁 전달
- 리스트형 콘텐츠

---

## 4. 스토리형 (Story Hook)

### 설명
개인 경험이나 상황으로 시작해 공감을 얻습니다.

### 패턴
```
"[과거 시점], 저는 [상황]이었어요."
"어제 [사건]이 있었는데요."
"[기간] 전만 해도 저는 [상태]였어요."
```

### 예시
```
✅ "6개월 전, 저는 매일 야근하는 직장인이었어요."
✅ "어제 클라이언트한테 '이거 언제 돼요?' 연락 받았는데요."
✅ "1년 전만 해도 저는 AI가 뭔지도 몰랐어요."
```

### 적합한 상황
- Before/After 공유
- 실제 경험 기반 콘텐츠
- 친밀감 형성

---

## 5. 비밀/독점형 (Secret Hook)

### 설명
숨겨진 정보나 인사이더 지식을 공유하는 느낌을 줍니다.

### 패턴
```
"아무도 안 알려주는 [주제]"
"[대상]만 아는 [비밀]"
"공개하기 아까운 [내용]"
```

### 예시
```
✅ "대기업에서 아무도 안 알려주는 보고서 작성법"
✅ "상위 1% 크리에이터만 아는 알고리즘 비밀"
✅ "공개하기 아까운 제 프롬프트 템플릿"
```

### 적합한 상황
- 전문 지식 공유
- 희소성 강조
- FOMO 유발

---

## 6. 대비형 (Contrast Hook)

### 설명
Before/After, 일반 vs 특별 등 대비를 통해 가치를 드러냅니다.

### 패턴
```
"[일반적 방법] vs [특별한 방법]"
"[Before 상태] → [After 상태]"
"남들은 [A], 저는 [B]"
```

### 예시
```
✅ "3시간 걸리던 작업 → 15분으로"
✅ "남들은 ChatGPT로 검색, 저는 ChatGPT로 자동화"
✅ "무료 vs 유료 AI 툴, 진짜 차이는?"
```

### 적합한 상황
- 효과/가치 증명
- 차별화 포인트 강조
- 업그레이드 제안

---

## 패턴 선택 가이드

| 콘텐츠 유형 | 추천 패턴 |
|------------|----------|
| 문제 해결 팁 | 질문형, 숫자형 |
| 인사이트/관점 | 충격형, 비밀형 |
| 경험 공유 | 스토리형, 대비형 |
| How-to 가이드 | 숫자형, 대비형 |
| 트렌드 분석 | 충격형, 질문형 |
| 도구/리소스 소개 | 비밀형, 숫자형 |

---

## 채널별 권장 패턴

### 링크드인
1순위: 스토리형, 숫자형
2순위: 질문형, 대비형

### X
1순위: 충격형, 숫자형
2순위: 비밀형, 질문형

### 뉴스레터
1순위: 스토리형, 비밀형
2순위: 질문형, 대비형

---

## 조합 사용 예시

### 질문 + 숫자
```
"왜 어떤 사람은 하루 2시간만 일해도 성과가 나올까요?"
```

### 충격 + 스토리
```
"열심히 일하면 성공한다는 말, 거짓말이었어요.
3년 동안 매일 야근했는데 승진은 남이 했거든요."
```

### 비밀 + 대비
```
"상위 1%만 아는 비밀:
일반인은 프롬프트를 '입력'하고, 그들은 '설계'해요."
```
