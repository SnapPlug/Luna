## X 알고리즘 이해

### For You 피드 구조
X의 For You 피드는 두 가지 소스로 구성됩니다:
- **Thunder (인네트워크)**: 팔로우한 계정의 최근 게시물
- **Phoenix (아웃오브네트워크)**: ML 기반으로 발견한 글로벌 게시물

### 점수 계산 방식
Grok 기반 트랜스포머 모델이 각 게시물의 참여도를 예측합니다:

**최종 점수 = Σ(가중치 × 각 행동의 확률)**

예측되는 행동 (가중치 높은 순):
1. 💬 **답글** - 가장 높은 가중치
2. 🔄 **리포스트** - 매우 높은 가중치
3. 💭 **인용** - 높은 가중치
4. ❤️ **좋아요** - 중간 가중치
5. 🖱️ **클릭/조회** - 기본 가중치
6. 👤 **프로필 클릭** - 기본 가중치
7. ➕ **팔로우** - 보너스 가중치

### 알고리즘 핵심 원칙
- 진정한 사용자 상호작용이 핵심
- 수작업 조작 불가능한 ML 기반 시스템
- 작가 다양성 점수로 피드 다양성 확보

## 알고리즘 최적화 전략

### 1. 답글 유도 (최고 가중치)
```
✅ 열린 질문으로 끝내기
   "여러분은 어떻게 생각하세요?"
   "다른 방법 있으면 알려주세요!"

✅ 논쟁적 의견 던지기
   "저는 A가 낫다고 생각하는데요"

✅ 경험 공유 요청
   "비슷한 경험 있으신 분?"
```

### 2. 리포스트 유도 (매우 높은 가중치)
```
✅ 저장/공유 가치 있는 정보
   - 체크리스트, 팁 모음
   - "나중에 쓸 것 같은" 정보

✅ 공감 포인트
   - 많은 사람이 "이거 맞아!" 할 내용
   - 대신 말해주는 느낌

✅ 인사이트 한 줄
   - RT해서 똑똑해 보이고 싶은 내용
```

### 3. 인용 유도 (높은 가중치)
```
✅ 의견 추가하고 싶게 만들기
   "이건 첫 단계일 뿐이에요"

✅ 반박하고 싶게 만들기
   "솔직히 [일반적 믿음] 과대평가됐어요"

✅ 사례 추가하고 싶게 만들기
   "저 경우엔 이랬는데..."
```

### 4. 클릭/조회 최적화
```
✅ 첫 줄에서 스크롤 멈추게
   - 숫자로 시작
   - 질문으로 시작
   - 충격적 statement

✅ 궁금증 유발
   "결과가 어땠을까요?"
   👇 아래에서 확인
```

## 작성 원칙

### 한 줄의 힘
- 첫 줄이 전부 (타임라인에서 보이는 부분)
- 스크롤 멈추게 하는 임팩트
- 호기심 유발 → 클릭 → 참여

### 핵심만 남기기
- 분석 결과에서 가장 강력한 포인트 1개 선택
- 불필요한 수식어 제거
- 짧고 강렬하게

### 참여 설계
- 항상 "다음 행동"을 유도
- 질문, 의견 요청, 공유 요청 중 선택
- 강요하지 않고 자연스럽게

### 톤
- 링크드인보다 캐주얼
- 반말도 가능
- 위트 있게
- 진정성 있게 (가짜 친근함 X)

## 효과적인 훅 예시

### 질문형 (답글 유도)
```
~하는 사람 손? 🙋
왜 아무도 이걸 안 알려줬지?
이거 나만 몰랐어?
```

### 숫자형 (신뢰도 + 클릭)
```
3가지만 기억하세요
10분이면 충분해요
87%가 모르는 사실
```

### 충격형 (인용/리포스트 유도)
```
솔직히 말할게요.
이거 알면 생각이 바뀜
진작 알았으면 좋았을 것
```

### 대비형 (공감 + 리포스트)
```
Before: 3시간
After: 15분

남들: 열심히
나: 똑똑하게
```

### 도발형 (인용 유도)
```
[일반적 믿음] 과대평가됐어요
불편한 진실 하나 말할게요
논쟁적일 수 있는데...
```

